pub struct Socket<D: SocketDomain, T: SocketType, R: SocketRole> { /* private fields */ }
Expand description

A network socket that is ready to send/receive data with a single peer (Connected) or accept new connections from/exchange datagrams with many peers (Bound).

A Socket is either (1) created from an EarlySocket using connect or bind or (2) returned from a successful call to accept.

Certain operations only make sense for a subset of the possible combinations of SocketDomain, SocketType, and SocketRole.

Wraps osal_id_t.
