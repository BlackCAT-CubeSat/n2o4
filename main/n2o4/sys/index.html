Expand description

Low-level, unsafe bindings to the C APIs of cFE and OSAL.

This is n2o4’s equivalent of a *-sys crate for the cFE and OSAL APIs. Generally, users of n2o4 will not use this module directly, preferring the safe wrappers provided in the rest of n2o4. However, this module is available if one needs to use something for which safe wrappers haven’t been made.

The definitions in this module are auto-generated by bindgen, so there’s no Rustdoc comments available. Check out the cFE user’s guide for details about types and functions.

Those identifiers starting with S_, X_, and SHIM_ are the result of hand-written C code required to make bindgen create bindings for certain constants and function; each is the same as the C constant or function without the prefix in its name.
