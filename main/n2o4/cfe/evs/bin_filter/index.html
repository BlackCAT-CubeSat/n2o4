Values intended for use in the Mask field of BinFilter.

Constants

  • EVERY_FOURTH_ONE
    Value for Mask: every fourth event message is sent.
  • EVERY_OTHER_ONE
    Value for Mask: every other event message is sent.
  • EVERY_OTHER_TWO
    Value for Mask: sends two messages, filters out two, then repeats.
  • FIRST_4_STOP
    Value for Mask: only the first four event messages are sent.
  • FIRST_8_STOP
    Value for Mask: only the first eight event messages are sent.
  • FIRST_16_STOP
    Value for Mask: only the first 16 event messages are sent.
  • FIRST_32_STOP
    Value for Mask: only the first 32 event messages are sent.
  • FIRST_64_STOP
    Value for Mask: only the first 64 event messages are sent.
  • FIRST_ONE_STOP
    Value for Mask: only the first event message is sent.
  • FIRST_TWO_STOP
    Value for Mask: only the first two event messages are sent.
  • NO_FILTER
    Value for Mask: all event messages are sent (until the event counter saturates).
